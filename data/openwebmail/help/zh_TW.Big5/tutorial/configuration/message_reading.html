OpenWebMail 說明指南 >> 讀信相關設定

讀信相關設定相當簡單，改變收信的方法只要確定方塊勾選或是不勾選就可以了。你可以依據你的使用偏好和最適螢幕寬度來設定。

讀信相關設定對照表

項目/設定 說明/改變
讀信時，使用固定寬度字型使用固定寬度的字型，某些人會覺得這樣比較漂亮
讀信時，使用笑臉圖示使用笑臉圖示(就像是用在e-mail或是信區的心情符號)顯示在郵件中。
關閉郵件內的 JavaScript關閉郵件內支援 JavaScript 的功能，避免惡意的信件透過 JavaScript 干擾你的 Browser
關閉郵件內的 Embedded CGI關閉郵件內支援 CGI 的功能，避免某些廣告信件透過此種方式確認使用者的讀信動作
以超連結方式顯示圖片附件以超連結方式顯示圖片附件，節省讀信所需的傳輸時間
外寄郵件編碼外寄郵件所使用的 字集
