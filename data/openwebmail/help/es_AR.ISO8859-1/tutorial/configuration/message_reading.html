Ayuda de OpenWebMail >> Opciones de Lectura de Mensajes

Las Opciones de Lectura de Mensajes son opciones simples que cambian la forma en que son mostrados los mensajes. Puede cambiarlas de acuerdo a sus preferencias para ajustarlas al tamaño de su pantalla y a su gusto personal.

Tabla de referencia:

Término/Opción Configura/Cambia
Mostrar mensajes con tipografía de ancho fijo Utilizar tipografía (fuente) de ancho fijo al mostrar el texto de un mensaje.
Usar "smiles" al mostrar mensajes Utilizar imágenes para mostrar "smiles" o "emoticons" dentro de los mensajes (p.ej.: :) ).
Desactivar JavaScript dentro de los mensajes Deshabilita la ejecución de código JavaScript dentro de mensajes HTML.
Desactivar CGI embebido dentro de los mensajes Deshabilita la ejecución de código CGI embebido dentro de mensajes HTML.
Mostrar imágenes adjuntas como enlaces Utiliza un enlace para mostrar imágenes adjuntas.
