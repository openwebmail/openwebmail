CGI

Abreviatura de Common Gateway Interface, una especificación para transferir información entre un servidor de la World Wide Web y un programa. Un programa CGI es cualquier programa diseñado para aceptar y retornar datos que conforman la especificación CGI. El programa puede ser escrito en cualquier lenguaje de programación, incluyendo C, Perl, Java o Visual Basic. Los programas CGI son la forma más común para que los servidores Web interactúen dinámicamente con los usuarios. Muchas páginas HTML que contienen formularios, por ejemplo, usan programas CGI para procesar los datos del formulario una vez que es enviado. Otra forma muy común de proveer interacción entre las páginas Web y los usuarios es incluir scripts o programas que se ejecutan en la máquina de éste último en vez de en el servidor Web. Estos programas pueden ser applets Java, Javascripts o controles ActiveX. Estas tecnologías se denominan soluciones "client-side", mientras que el uso de CGI es una solución "server-side" dado que el procesamiento ocurre en el servidor Web.
