Mapa de Carateres

Una lista definida de caracteres reconocidos por el hardware y el software de la computadora. Cada caracter (letra, símbolo o número) es representado por un número. El mapa de caracteres ASCII, por ejemplo, utiliza números desde el 0 al 127 para representar todos los caracteres del idioma inglés, así como también caracteres especiales de control. El mapa de caracteres ISO Europeo estándar es similar al ASCII pero contiene caracteres adicionales para los idiomas europeos.
