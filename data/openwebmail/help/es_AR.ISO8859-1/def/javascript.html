Javascript™

Un lenguaje de scripting desarrollado por Netscape para facilitar la construcción de sitios interactivos. Aunque comparte varias de las características y estructuras del lenguaje Java, fue desarrollado independientemente del mismo. Javascript puede interactuar con código fuente HTML, permitiendo el agregado de contenido dinámico a los sitios web. Javascript es soportado por un gran número de empresas productoras de software y es un lenguaje abierto que cualquiera puede utilizar sin comprar una licencia. Es soportado por los navegadores más recientes desde Netscape hasta Microsoft, aunque Internet Explorer solo soporta un subconjunto del lenguaje que Microsoft llama "Jscript".
