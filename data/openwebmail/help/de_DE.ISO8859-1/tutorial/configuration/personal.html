OpenWebMail Help Tutorials >> Persönliche Einstellungen

In den Persönlichen Einstellungen wird gesetzt, was andere beim Erhalt einer Email vom Absender sehen sowie andere Einstellungen.

Persönliche Einstellungen Tabelle :

Einstellung Auswirkung
Sprache Anzeigesprache und Zeichensatz können hier eingestellt werden.
Zeitzonenabstand (zu GMT): Die Zeitzone im Verhältnis zur GMT.
Sommerzeit Sommerzeit Einstellung
Von Die E-Mailadresse, die im Von Feld angezeigt wird.
Antwortadresse Hier kann man die Antwortadresse eingeben, wenn Sie von der Adresse un Von-Feld abweichen sollte.
Nachrichten automatisch weiterleiten an Hier kann eine automatische Weiterleitung eingestellt werden.
Automatische Antwort Hier kann eingestellt werden, dass eine automatische Antwort gesendet wird.
Text für automatische Antwort (Max. 500 Zeichen) Hier kann der Text für die automatische Antwort eingegeben werden.
Signatur (max. 500 Zeichen)
 Hier kann die persönliche Signatur gesetzt werden.
