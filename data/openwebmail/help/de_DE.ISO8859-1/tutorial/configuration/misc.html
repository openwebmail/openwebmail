Hier sind Einstellungen, die nicht in eine der anderen Gruppen passen .

Verschiedene Einstellungen Tabelle :

Einstellung Auswirkung
Standardzeichensatz für das Dateisystem Wählen Sie den Zeichensatz für das Dateisystem
Leuchtbalken zum Auswählen benutzen Ja/Nein
Regular Expressions beim Suchen und für Filterregeln verwenden Ja/Nein
Interne Ordnerdaten verbergen Ja/Nein
Kategorisierte Ordnerliste Auflistung der Userordner nach Kategorien. Dafür muss der Ordner als Kategorieordner definiert sein. Hier kann auch das Trennzeichen für die Auflistung der Ordner definiert werden.
Akkustische Benachrichtigung Hier kann die Wavedatei gewählt werden, die bei Eingang von Neuen Nachrichten abgespielt wird.
Benachrichtigung bei neuen Nachrichten durch Popup-Fenster Hier kann die Anzeigedauer des Popup-Fensters eingestellt werden. 0 schaltet das Fenster ab.
Benachrichtungsfenster "Nachricht gesendet": Hier kann die Anzeigedauer des Popup-Fensters eingestellt werden. 0 schaltet das Fenster ab.
Wörterbuch für die Rechtschreibprüfung Das Defaultwörterbuch zur Rechtschreibprüfung kann hier gesetzt werden. Ein anderes Wörterbuch kann "on the Fly" für eine Nachricht geändert werden.
Gelöschte Nachrichten maximal halten (Tage) Anzahl der Tage, wie lange die gelöschte n Nachrichten aufgehoben werden.
Spam/Virus-Nachrichten maximal halten (Tage): Anzahl der Tage, wie lange die SpamMails aufgehoben werden.
Intervall zur Aktualisierung Minuten bis zur Überprüfung auf neue Nachrichten.
Ablauf der Sitzung nach Stunden bis zum Ablaufen der Sitzung.
