OpenWebMail Help Tutorials >> Speichern einer Nachricht

Sie wollen eine Nachricht erstellen und später senden? Dies kann bewerkstelligt werden, indem Sie eine Nachricht als Entwurf abspeichern. Alle Daten der Nachricht werden gespeichert (An, Von, Betreff etc.), auch die Nachricht an sich wird gespeichert. Haben Sie die Nachricht geschrieben, klicken Sie auf "Als Entwurf speichern", die Nachricht wird unter dem Ordner "Entwürfe" gespeichert. Um die Nachricht danach zu senden, öffnen Sie sie einfach aus Ihrem Entwürfe-Ordner, (siehe Verwalten von Ordnern ), und klicken Sie "Senden". Sie können die Nachricht auch vor dem Senden bearbeiten.
