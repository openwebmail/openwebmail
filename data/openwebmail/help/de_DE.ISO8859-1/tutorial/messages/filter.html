OpenWebMail Help Tutorials >> Einen Filter einstellen

Filter können unerwünschte Nachrichten blockieren. Der Trick ist es, einen Filter zu erstellen, der bestimmte Mails betrifft, andere aber nicht. Ist der Filter definiert, geht es daran, eine Aktion zu definieren, um mit diesen Mails entsprechend zu verfahren.

Um einen Filter anzulegen, klicken Sie "Mail-Filter-Einstellungen" , die Seite zur Filter-Einstellung wird angezeigt. Sie werden feststellen, dass bereits einige globale Filter gesetzt sind. Diese Filter sind nicht löschbar. Es ist durchaus möglich, dass sie mit diesen Filtern schon auskommen. Es werden zuerst die eigenen Filter ausgeführt, dann die globalen Filter. Dadurch können Sie eventuell die globalen Filter umgehen.

Wollen Sie jedoch einen weiteren Filter setzen, finden Sie hier die Möglichkeit, neue Filter zu setzen sowie bestehende Filter zu ändern und zu löschen.

Wie Sie sehen, sind einige Filter gesetzt, allerdings ist unklar, auf welche Mails sie wirken. Z.B. könnte ein Filter so aussehen: \s{40,}\[[a-z0-9]{5}\]$. Dies ist ein spezieller Filter, der auf mehrere Wörter zutreffend ist. In diesem Fall trifft alles zu, das 40 oder weniger klein geschriebene Zeichen und 5 oder weniger alphanumerische Zeichen beinhaltet. Wie auch immer, dieses Tutorial kann nur auf das grundlegende Setzen von Filtern eingehen, Details finden Sie im Internet unter Regular Expressions.

Verwenden Sie diese Tabelle als Referenz für Mailfilter :

Bezeichnung Aktion oder Verwendung
Regel wählt den Header-Teil, auf den der Filter Anwendung finden soll
Inkl/Exkl beinhaltet oder beinhaltet nicht folgende Wörter
String Text, der blockiert oder akzeptiert werden soll
Aktion die Aktion, die ausgeführt werden soll, wenn der Filter zutrifft
Zielordner Ordner, in den die Nachricht verschoben werden soll
aktivieren schaltet den Filter aktiv
Action[2] abschließende Filteraktionen (hinzufügen/bearbeiten)

Dieser Filter bedeutet auf Deutsch:
Verschiebe alle Nachrichten Von jedem auf der Site bad-website.com in den Papierkorb. Dieser Filter ist aktiv. Daher wird jede Mail, die von jemanden mit der Adresse bob@bad-website.com kommt, in den Papierkorb verschoben.

Sie können auch einen exkludierenden Filter setzen. Dieser blockiert dann alles, das dem Filter nicht entspricht. Wenn Sie beispielsweise in dem oberen Filter inkludieren auf exkludieren setzen, erlauben Sie ausschließlich Mails, die von bad-website.com kommen.

Um einen Filter zu bearbeiten, klicken Sie einfach auf den String des Filters. Dann wird der Filter oben im Bearbeitungsbereich angezeigt. Zur Bearbeitung von Filtern siehe folgende Erklärungen.

Wollen Sie kurzfristig einen Filter ausschalten, gibt es einen einfacheren Weg , als den Filter zu löschen und danach wieder einzugeben. Folgen sie den Arbeitsschritten, um einen Filter zu bearbeiten, und ändern Sie nun die Checkbox "aktivieren", klicken sie dann auf Hinzufügen/Bearbeiten, danach ist der Filter nicht mehr aktiv. Die inaktiven Filter werden in einer eigenen Tabelle dargestellt. Um den Filter wieder zu aktivieren, wählen Sie nun den Filter nochmals aus und aktivieren Sie die Checkbox aktivieren. Danach durch Hinzufügen/Bearbeiten die Änderung bestätigen, und der Filter ist wieder aktiv. Wollen Sie den Filter löschen, können Sie im inaktiven Zustand den Button Löschen klicken (rechts in der Zeile des entsprechenden Filters).
