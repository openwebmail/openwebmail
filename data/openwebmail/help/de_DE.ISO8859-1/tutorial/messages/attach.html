OpenWebMail Help Tutorials >> Anhänge hinzufügen

Sie können einen Anhang der E-Mail hinzufügen. Ein Anhang ist eine Datei, die in der E-Mail nicht direkt angezeigt werden kann. Das erlaubt es dem Empfänger, den Anhang zu bekommen und ihn in einem anderen Programm zu betrachten. Allerdings muß der Empfänger ein entsprechendes Programm zur Darstellung besitzen.  Anhänge können sinnvoll sein, um das Format eines Dokuments beizubehalten.  DOC ist ein gängiges Format (Microsoft Word Dokument oder Windows WordPad Dokument), aber auch RTF (Rich Text Format).  Sie können auf diesem Weg auch Bilder mitschicken. Versichern Sie sich zuvor aber, ob der Empfänger diese Dokumente auch anzeigen kann.

Um einen Anhang hinzuzufügen, drücken Sie "Durchsuchen" in der Zeile Anhang. Dies öffnet eine Dialogbox zum Auffinden der Datei. Wählen Sie die Datei aus, drücken Sie in der Dialogbox öffnen und danach Hinzufügen, um den Anhang der Mail hinzuzufügen. Seien sie bitte geduldig beim Hinzugügen des Anhanges, je nach Größe der Datei und der Geschwindigkeit Ihrer Internetverbindung kann das Berechnen einige Zeit in Anspruch nehmen.  Beachten Sie bitte, dass die Datei erst angehängt wird, wenn Sie Hinzufügen klicken. Dieser Vorgang kann beliebig oft wiederholt werden. Allerdings kann es Server geben, die zu große Mails blockieren.

Um einen Anhang aus dem Mail zu entfernen, klicken Sie auf "Löschen" bei dem entsprechenden Anhang.
