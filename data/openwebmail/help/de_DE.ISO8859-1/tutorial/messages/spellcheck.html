OpenWebMail Help Tutorials >> Benutzung der Rechtschreibprüfung

Haben Sie eine Nachricht geschrieben, wollen Sie diese möglicherweise vor dem Senden auf Rechtschreibfehlerüberprüfen. Webmail kann dies mit seiner eigenen Rechtschreibprüfung erledigen. Wählen Sie einfach das entsprechende Wörterbuch (Deutsch, English oder American) und klicken Sie "Rechtschreibprüfung". Ein neues Fenster mit der Rechtschreibprüfung geht auf.

Lautet die Nachricht:

Dies ist eine Testmail

bekommen Sie eine Rechtschreibprüfung die so aussieht :

Die falsch geschriebenen Wörter sind die Wörter in den Dropdown-Menüs. Wählen Sie die Wörter, die Sie ändern wollen. Eine Liste von vorgeschlagenen Wörtern wird präsentiert; es ist allerdings auch möglich, manuell eine Änderung zu machen. Sie können eines der vorgeschlagenen Wörter nehmen oder -- Manuelle Korrektur -- wählen, wenn das Wort nicht in der Liste ist. Wenn Sie fertig sind, beenden Sie die Rechtschreibprüfung. Das Fenster wird geschlossen, und die Änderungen werden automatisch in den Text übernommen. Haben Sie -- Manuelle Korrektur -- gewählt, wird der Bereich angezeigt, in dem der Fehler ist, und Sie können die Änderung vornehmen. Haben Sie die Änderung vorgenommen, können Sie die Rechtschreibprüfung beenden, und Sie kommen zur geänderten Nachricht zurück.
