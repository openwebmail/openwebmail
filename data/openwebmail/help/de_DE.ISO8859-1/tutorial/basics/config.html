OpenWebMail Help Tutorials >> Benutzer Grundeinstellungen

Die Grundeinstellungen erlauben es, grundlegende Einstellungen zu Ihrem E-Mailclient zu machen. Wenn Sie das erste Mal Webmail benutzen, werden Sie durch die Einstellungen geleitet, um die Oberfläche und die Inbox Einstellungen zu machen. Wie auch immer, Sie wollen Ihren E-Mailclient so schnell wie möglich funktionsbereit haben. Dieses Tutorial soll beschreiben, wie sie die Grundeinstellungen setzen können.

Das sind die Hauptpunkte der Einstellungen :

  • Persönliche Einstellungen
  • Darstellungs-Einstellungen
  • Lese-Einstellungen
  • Nachrichten-Einstellungen
  • Antwort-Einstellungen
  • Filter-Einstellungen
  • Kalender-Einstellungen
  • Weitere Einstellungen

Für die meisten sind die Grundeinstellungen ausreichend, sollte das für Sie ebenfalls zutreffen, drücken Sie speichern.

Ihr System Administrator kann Ihre Einstellungen unter Umständen überschreiben, wenn er dies für nötig hält.
