OpenWebMail Help Tutorials >> Verwendung von Adressbucheinträgen

Natürlich wollen Sie die in Ihr Adressbuch eingegebenen Daten auch benutzen. Öffnen Sie eine neue Mail (Siehe schreiben einer Mail ). Sie werden Headereinträge sehen, die Sie ändern können (An, Betreff , etc.).

Sie können schnell und einfach Adressen aus Ihrem Adressbuch in das An Feld einfügen. Klicken Sie einfach auf das "Adressbuch-Icon" ( ) nahe des An Feldes. Ein neues Fenster wird geöffnet, mit allen Einträgen im Adressbuch versehen, mit einer Checkbox bei jedem Eintrag. Wählen Sie, mit Hilfe der Checkboxen, die Einträge, die Sie übernehmen wollen, und wählen Sie "weiter". Die Adressen werden nun in dem entsprechenden Feld, das Sie gewählt haben, eingefügt (in diesem Fall das An Feld). Sie können genau so Adressen in das CC oder BCC Feld einfügen, indem Sie das korrespondierende Adressbuch Icon klicken. Klicken Sie "Cancel", wird das Adressbuch geschlossen, und der Einfügevorgang wird abgebrochen.

Sind Sie bereits im Adressbuch, können Sie einen Eintrag anklicken, und es wird automatisch eine neue Mail mit der entsprechenden Adresse geöffnet. Das ist der zweite Weg, eine Adresse in eine neue Mail einzufügen.
