SMTP (Simple Mail Transer Protocol)

Kurz für Simple Mail Transfer Protocol, ein Protokoll, Mails zwischen Servern zu transportieren. Die meisten Mail-Systeme können Mails via SMTP von einem Server im Internet zu einem anderen Server transportieren. Die Nachrichten können dann von einem E-Mailclient via POP oder IMAP abgeholt werden. Zusätzlich wird SMTP verwendet, um von einem E-Mailclient Mails zu einem Server zu transportieren. Das ist der Grund, warum man beides, POP- oder IMAP-Server, und den SMTP-Server angeben muß, wenn man seinen E-Mailclient einstellt.
