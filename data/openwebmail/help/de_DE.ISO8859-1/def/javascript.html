Javascript™

Dies ist eine Skriptsprache, eingeführt von Netscape, um dem Webautor zu erlauben, interaktive Seiten zu gestalten. Javascript ist ein Auszug aus der kompletten Programmiersprache Java. Javascript kann mit dem HTML-Sourcecode interagieren und ist damit geeignet, dynamische Webseiten zu kreieren. Javascript ist eine Openlanguage und kann dadurch verwendet werden, ohne Lizenzgebühren zu bezahlen. Javascript wird von vielen Webbrowsern unterstützt, Microsofts Internet Explorer unterstützt nur einen Teil der Sprache und nennt dies Jscript.
