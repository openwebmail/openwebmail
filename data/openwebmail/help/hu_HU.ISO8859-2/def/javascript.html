A Netscape által kifejlesztett szkriptnyelv az interaktív weblapok megvalósításához. Struktúrájában hasonló, és néhány eleme azonos a Java nyelvvel, önállóan fejlesztették ki. A javascript interakcióba léphet a HTML forrással, lehetõvé téve a web fejlesztõknek dinamikus tartalom használatát. A JavaScriptet sok szoftver cég használja, de megmaradt szabad nyelvnek, azaz bárki írhat benne programot, használhatja licensz feltételek és kiadások nélkül. Támogatott a nagyobb böngészõkben, így a Netscape, illetve a Mozilla alapúakban, és az Internet Explorerben, bár ez utóbbi csak egy részét, amit a Microsoft Jscript-nek hív.
