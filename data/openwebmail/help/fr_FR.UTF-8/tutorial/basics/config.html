Tutoriel d'aide OpenWebMail >> Configuration des réglages de base

La configuration des réglages de base vous permet de modifier de nombreuses préferences quant à vos besoins concernant votre messagerie élevctronique. Si c'est la première fois que vous utilisez OpenWebMail, vous êtes automatiquement dirigé vers le panneau de contrôle de configuration des réglages plutôt que vers votre boîte de réception. Cependant, vous voudrez probablement rendre votre outil OpenWebMail utilisable aussitôt que possible. Ce tutoriel permet d'effectuer la configuration de base de vos réglages.

Voici les points principaux relatifs à la configuration de vos réglages:

  • Informations personnelles
  • Préferences d'affichage
  • Options de lecture des messages
  • Utilisation des messages
  • Options de réponse aux messages
  • Options de filtrage des messages
  • Options de calendrier
  • Options diverses

Pour de nombreux utilisateurs, les réglages proposés par défaut sont corrects. Si vous ne souhaitez pas modifier ces réglages maintenant, cliquez sur le bouton "Enregistrer" en bas de la page.

Votre administrateur système a probablement préconfiguré ou supprimé certaines de ces options pour vous.
