Tutoriel d'aide OpenWebMail >> Lire un message

Pour lire un message, repérez le message dans la liste affichée de votre Boîte de réception , et cliquez sur le sujet du message.

Ceci vous dirigera vers un écran similaire à celui présenté à main droite:

La barre d'outils de messagerie permet d'effectuer des actions sur le message.

L' En-tête de message contient les mêmes informations que celles affichées dans la Boîte de réception . Cliquez sur En-tête complet pour afficher les informations détaillées correspondant à l'en-tête.

Le message proprement dit constitue le texte que l'expéditeur vous a envoyé.

L' Index de message ( flèches gauche et droite de part et d'autre d'un numéro) situé sous la barre d'outils, permet de passer rapidement d'un message à l'autresans revenir aux dossiers/listes de la boîte de réception.
