Tutoriel d'aide OpenWebMail >> Barre d'outils de recherche

La barre d'outils de recherche vous permet de rechercher des messages dans le dossier courant.

Utiliser cette barre est plutôt simple. La Double flèche gauche ( )  permet d'accéder à la première page de messages du dossier courant. Voir Préférences d'affichage pour modifier le nombre de messages affichés par page.

La flèche gauche ( ) permet d'accéder à la page de messages précédente.

Le champ texte "1", indique le numéro de la page courante sur le nombre total de pages. Vous pouvez modifier cette valeur pour accéder à n'importe quelle page de messages dans le dossier. Par exemple, s'il est indiqué "[Page 2 de 7]", vous pouvez taper "5" à la place de "2", valider par un appui sur la touche "Entrée" au clavier et ainsi atteindre directement la page 5.

La flèche droite, et la Double flèche droite font exactement l'opposé des flèches correspondantes à gauche.

Vous pouvez rechercher les messages dans le dossier courant en tapant simplement votre requête dansle champ texte à coté de "Chercher", et en sélectionnant le critère de recherche à préciser à l'aide du bouton de liste de gauche (Sujet, Message etc...). Ainsi, vous pouvez rechercher parmi les expéditeurs, les destinataires, le Sujet, la Date, les noms de fichiers attachés, Tous les éléments ci-avant (En-tête), le contenu du Message, ou bien tout ( Tous ). Une fois vos critères précisés, cliquez sur le bouton "Chercher", ou frappez "Entrée" sur le clavier.

Cette barre d'outils est destinée à la recherche de messages dans sa forme la plus basique, et ce dansla boîte aux lettres ou le dossier courant.  Vous disposez d'une barre d'outils de recherche avancée poureffectuer des recherches plus détaillées.  Veuillez consulter Recherche avancée pour apprendre comment réaliser des requêtes de recherche spécifiques.
