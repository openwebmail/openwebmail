Tutoriel d'aide OpenWebMail >> Options diverses

Les options diverses regroupent un certain nombre de réglages qui n'ont pu à priori être rattachésà telle ou telle catégorie de réglages.

Eléments de référence:

Terme/Réglage Ce qui est positionné/modifié
Notification sonore de nouveau(x) message(s)Emet un son à l'arrivée d'un nouveau message, le son joué correspondant au fichierson choisi avec le bouton de liste à droite.
Utiliser les expressions régulières dans les recherches et les filtresUtilise les Expressions régulières pour la fonction de recherche et les règles de filtrage.
Masquer les informations systèmeMasque les données système internes de l'onglet.
Categorized Folders listOn the folders page, list user folders by category. You define the category by naming the folder as category-foldername, separated by the string or character of your choice (default is "-", "/" is not allowed)
Dictionnaire de correction orthographiqueLe dictionnaire par défaut à utiliser lorsqu'on contrôle l'orthographe d'un message (peut être modifié "A la volée").
Conserver les messages dans la Poubelle (jours)Nombre de jours pendant lesquels les E-mails contenus dans le dossier "Poubelle" seront conservés. Après le nombre de jours spécifié, le dossier "Poubelle" sera vidé.
Temps limite d'inactivitéTemps maximum alloué à votre session OpenWebMail sans action de votre part.
Intervalle de rafraîchissementNombre de minutes entre les tentatives de rappatriement de nouveaux E-mails.
Fenêtre de notification de nouveau messageNombre de secondes pendant lesquelles la fenêtre Pop-Up de notification de nouveau message reste affichée.  La valeur "0" inhibe cette fonction.
