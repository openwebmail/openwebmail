Tutoriel d'aide OpenWebMail >> Options de réponse aux messages

Les options de réponse aux messages règlent la manière dont il est répondu aux messages.

Eléments de référence:

Terme/Réglage

Ce qui est positionné/modifié
Jeu de caractères pour la réponseCeci permet de sélectionner le jeu de caractères à utiliser pour la réponse.
Reparagrapher le message avant d'y répondreReparagraphe un message avant que vous y répondiez.
Insérer le message original lors d'une réponsePermet de sélectionner la position à laquelle placer le message d'origine lors de la réponse.
Sauvegarder la réponse dans le dossier "Envoyés"Enregistre une copie de toutes les réponses dans le dossier "Envoyés".
