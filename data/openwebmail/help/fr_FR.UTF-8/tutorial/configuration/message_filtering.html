Tutoriel d'aide OpenWebMail >> Options de filtrage des messages

Le filtrage des messages détermine comment et où bloquer les messages à filtrer.

Eléments de référence:

Terme/Réglage Ce qui est positionné/modifié
Nombre de messages identiques avant mise à la poubellePositionne le filtre pour limiter les messages redondants.
Filtrer les messages avec un faux SMTPFiltre les messages qui présentent une fausse adresse SMTP .
Filtrer les messages avec un faux champ From (expéditeur)Filtre les messages qui comportent une fausse valeur dans le champ From (expéditeur).
Filtrer les messages avec de faux contenus EXEFiltre les messages comportant des applications EXE à priori suspectes.
