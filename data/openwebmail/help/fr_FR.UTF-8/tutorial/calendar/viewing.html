Tutoriel d'aide OpenWebMail >> Consulter le Calendrier

Le calendrier offre une fonctionnalité complète dans OpenWebMail.Cet outil vous permet de programmer des évènements et des rappels sur une base journalière. Pour consulter le calendrier, cliquez sur le bouton "Calendrier" ( ).
Vous serez alors dirigé vers l'écran suivant:

Le jour courant est mis en évidence par un gris plus clair que les autres jours du mois.

Pour visualiser un jour spécifique, vous pouvez cliquer sur son numéro. Ceci vous dirigera vers les évènements et heuresde ce jour. Vous noterez qu'il existe une barre d'outils spécifiques pour le calendrier. Consultez Barre d'outils de calendrier pour plus d'informations.
