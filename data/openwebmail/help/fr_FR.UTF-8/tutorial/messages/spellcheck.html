Tutoriel d'aide OpenWebMail >> Utiliser le correcteur d'orthographe

Vous composez votre message à l'intérieur de la zone de texte principale sur la fenêtre de composition de message. Une fois que vous avez terminé, vous pouvez souhaiter rechercher les fautes d'orthographe. OpenWebMail peut utiliser un vérificateur d'orthographe pour vous. Sélectionnez simplement le dictionnaire à utiliser (English, ou French ou un autre suivant le cas), et cliquez sur le bouton "Vérif. Orth.". Cela ouvrira une nouvelle fenêtre qui est la page du vérificateur d'orthographe.

Si le message était:

Salu george tu vas bien ? que penses tu de cet esai ?

Vous obtiendrez un écran de vérification similaire à l'écran suivant:

Les mots mal orthographiés dans le message sont affichés dans les menus déroulants. Pour corriger un mot, cliquez sur le menu déroulant correspondant à ce mot. Une liste de suggestions apparaît, de même que l'option -Correction manuelle-. Vous pouvez choisir un des mots proposés dans la liste, ou bien choisir -Correction manuelle-, si la solution n'est pas dans la liste. Une fois que vous avez terminé, cliquez sur "Fin vérif.". La fenêtre sera fermée, et les mots corrigés seront automatiquement remplacés. Si vous avez choisi -Correction manuelle-, cliquez sur "Vérif. encore", et vous serez en mesure de frapper dans les zones de texte qui s'afficheront là où se trouve(nt) le(s) mot(s) erroné(s). Une fois que chaque mot aura été corrigé manuellement, vous pourrez cliquer sur "Fin vérif." et vous vous retrouverez sur la page de rédaction de message, avec celui-ci corrigé.
