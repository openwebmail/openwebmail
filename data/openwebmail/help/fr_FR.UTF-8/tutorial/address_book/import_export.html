Tutoriel d'aide OpenWebMail >> Importer & Exporter le carnet d'adresses

Vous disposez probablement déjà d'un carnet d'adresses stocké sur votre ordinateur, mais vousne souhaitez pas perdre de temps à recréer toutes ses entrées dans votre carnet d'adresses OpenWebMail. C'est dans ce but que vous disposez de l'outil d'importation. Cliquez sur le bouton Importer adresses dans la barre d'outils du carnet d'adresses ( ) Vous serez alors redirigé vers la fenêtre d'importation. Cet outil est compatible avec Outlook Express 5 (et supérieurs), et Netscape Mail 4.x (où "x" correspond à la sous-version). Vous aurez besoin d'ouvrir votre logiciel client de messagerie "hors-ligne", pour en exporter le carnet d'adresses en utilisant le format de fichier CSV (Comma Separated Values) ou bien un fichier TXT (Fichier texte ASCII). Ces instructions sont précisées sur l'écran de l'outil d'importation du carnet d'adresses. Vous aurez besoin d'exporter le carnet d'adresses de votre logiciel de messagerie "hors-ligne" afin de l'importer dans le carnet d'adresses d'OpenWebMail (Veuillez consulter la documentation de votre logiciel client de messagerie pour la fonction exportation du carnet d'adresses de celui-ci). Une fois que vous avez exporté proprement et au bon format de fichier, ce carnet d'adresses , vous pouvez l'importer dans celui d'OpenWebMail. Pour faire cela, cliquez sur "Parcourir", et ouvrez le fichier CSV ou TXT dans le dossier de votre ordinateur où vous l'avez enregistré. Une fois le bon fichier sélectionné, cliquez sur "Ouvrir". Finalement, cliquez sur "Importer", et votre carnet d'adresses "hors-ligne" sera dupliqué dans le carnet d'adresses d'OpenWebMail.

Vous pouvez également réaliser la procédure opposée en exportant votre carnet d'adresses OpenWebMail. Pour réaliser cela, cliquez sur le bouton "Exporter adresses" ( ).Cela affichera vos adresses listées en format de texte standard dans une fenêtre dédiée à cet effet. Utilisez alors Fichier > Enregistrer sous... pour sauvegarder ce carnet d'adresses. Sélectionnez Fichier texte (*.txt) pour le Type. Vous pourrez alors importer ces entrées dans le carnet d'adresses de votre logiciel client de messagerie. Voir la documentation de celui-ci pour les instructions correspondantes...
