Tutorial OpenWebMail >> Download & Backup

Backup adalah data yang telah ditiru ke lokasi yang lain untuk menjaga keamanan file tersebut. Anda harus melakukan ini dengan semua file-file dan folder-folder yang sangat penting di dalam komputer anda pada waktu tertentu, apabila ada masalah yang tidak terduga mungkin akan terjadi pada komputer anda.

Anda hanya diberi jumlah ruang tertentu untuk menyimpan file anda di dalam account anda, termasuk pesan email di account OpenWebMail anda. Oleh karena itu, anda akan cepat kehabisan ruang (jika anda sudah mencapai batas jumlah maksimum dari ruang yang sudah diberikan, anda akan menerima email yang memberitahukan anda mengenai hal tersebut). Hal ini membutuhkan anda untuk memindahkan pesan-pesan anda dari OpenWebMail.  Bagaimanapun, anda mungkin memiliki pesan yang penting yang ingin anda simpan untuk waktu yang akan datang. Anda dapat mendownload pesan email anda ke dalam komputer anda dan melihatnya setiap saat dengan berbagai teks editor yang anda miliki.

Untuk men-download Folder Email Anda:

Anda dapat men-download pesan email anda dari folder OpenWebMail ke dalam komputer anda di dalam folder manager (klik tombol toolbar ). Ketika anda berada di dalam folder manager, anda dapat meng-klik ikon disk ( ), dan folder yang ingin didownload oleh anda. Dan kemudian file tersebut akan mulai didownload ke dalam komputer anda. Anda akan ditanya apakah anda ingin menyimpan file tersebut atau hanya ingin membuka file. Ketika ditanya, pilih tombol "Save...". Kemudian file browser kecil akan muncul. Lihat folder yang berada di dalam komputer anda dimana anda ingin menyimpan file tersebut, dan klik tombol "Save".

File-file tersebut disimpan dalam format .zip. File * .zip berisi file atau beberapa file yang telah dikompresi ke dalam satu file.  Anda memerlukan file kompresi/dekompresi guna membuka file-file ini. Berikut ini adalah program yang dapat anda gunakan untuk mengkompresi dan dekompresi file .zip (harap dicatat bahwa file ini sama sekali tidak berhubungan dengan with OpenWebMail):

  • Winzip
  • PicoZip
  • Free Zip Program
  • ZipWave

Untuk melihat Pesan Email Anda:

Sekali anda sudah men-download folder pesan email, anda dapat menggunakan salah satu dari program di atas untuk meng-dekompresi file tersebut dan membacanya. Anda tidak dapat meng-klik dua (2) kali file dekompresi ini untuk dibuka  Sebagai gantinya, bukalah teks editor (seperti Windows Notepad**), dan membukanya disini dengan menggunakan perintah Open di dalam menu File. Pesan-pesan dari folder yang anda download semuanya akan berada dalam format teks sederhana, bersama dengan semua catatan atas lengkap email anda.

Pesan-pesan email yang akan anda download, sayangnya, akan menjadi baris program komputer yang kacau yang tidak ingin anda lalui dengan susah payah apabila anda sedang kehabisan ruang untuk menyimpan pesan-pesan email anda di dalam OpenWebMail. Jika anda kehabisan ruang untuk menyimpan pesan email anda di dalam OpenWebMail, tetapi anda ingin tetap memilikinya agar tetap mudah digunakan, silahkan hubungi sistem administrator anda untuk menanyakan mengenai pembelian ruang disk pada account pengguna anda.

*Catatan: Jika anda menggunakan Microsoft Internet Explorer 5.5, anda mungkin akan mengalami kesulitan-kesulitan di dalam membuka file setelah men-download file tersebut. Hal ini akan ditampilkan dalam format file.pl. Jika demikian, ubahlah format file .pl menjadi .zip dan ikutilah instruksi di atas.

*Notepad sangat terbatas dengan bagaimana besarnya file tersebut dapat dibuka. Jika file tersebut terlalu besar, anda akan diminta untuk membuka file tersebut di dalam Wordpad.  Ketika anda diminta untuk melakukan hal tersebut, klik tombol "Yes".
