CGI (Common Gateway Interface)

Singkatan dari Common Gateway Interface, spesifikasi untuk memindahkan informasi antara server World Wide Web dan program CGI. Program CGI program adalah program manapun yang dirancang untuk menerima dan mengembalikan data yang dapat menyesuaikan dengan spesifikasi CGI. Program tersebut dapat ditulis dengan menggunakan bahasa program manapun, termasuk C, Perl, Java, atau Visual Basic. Program CGI adalah cara yang paling umum untuk Web server saling berhubungan dengan dinamis dengan para pengguna. Banyak halaman HTML yang berisi form, sebagai contoh, menggunakan program CGI untuk memproses form data ketika data tersebut diberikan. Cara lain yang umumnya meningkat adalah untuk menyediakan hasil balikan untuk pengguna Web termasuk script atau program yang dapat dijalankan di komputer pengguna dibandingkan denga Wev server. Program ini dapat berupa Java applets, Java scripts, atau ActiveX controls. Teknologi ini dikenal bersama sebagai solusi pada sisi client, ketika kegunaan dari CGI adalah solusi pada sisi server karena pemrosesan terjadi pada Web server.
