Character Set

Daftar karakter yang sudah ditetapkan dan dikenal oleh komputer hardware dan software. Setiap karakter diwakili dengan angka. Set Karakter ASCII, sebagai contoh, menggunakan angka dari 0 sampai dengan 127 untuk mewakili semua karakter English sama seperti karakter kontrol yang khusus. Set Karakter European ISO serupa dengan ASCII, tetapi mereka berisi karakter tambahan untuk bahasa Eropa.
