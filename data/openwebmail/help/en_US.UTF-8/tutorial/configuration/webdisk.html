OpenWebMail Help Tutorials >> WebDisk Options
The WebDisk Options set the formats and preferences for the webdisk feature.

See the following chart for the webdisk options:

Term/Setting What it Sets/Changes
Number of items in directory listThe number of items be showed in the directory list of the webdisk.
Confirm before directory/file move/copy
Display a confirm window before a directory/file is moved/copied in the webdisk
Confirm before directory/file deletion
Display a confirm window before a directory/file is deleted in the webdisk.
Confirm before directory/file compress
 Display a confirm window before directory/file is compressed
(zip, tgz, gzip) in the webdisk.
Size for file edit window
Sets the width and the height of the edit window.
