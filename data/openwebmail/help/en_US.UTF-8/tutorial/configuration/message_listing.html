OpenWebMail Help Tutorials >> Message Listing
Message Listing options set the preferences for the folder view.

Message Listing reference chart:

Term/Setting

What it Sets/Changes
Controlbar position in message list
 Control bar position at top or bottom in the folder list view.
Messages per page
This sets the maximum number of messages to list at a time.
Field order
 The order of message attributes fields in folder browsing.
Default sort
Default message sorting.
Default search
 Controls the search type that gets displayed in the pulldown next to the search field.
Timestamp used for message date
Default datetype (sent or received) used for date field in message list.
Display mini search icon
 Whether to use mini search icon in sender and subject fields in the message list.
