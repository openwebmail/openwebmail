OpenWebMail Help Tutorials >> Spam Check Options
The Spam Check Options set the preferences of the external spam checker (SpamAssassin). The spam checker in OpenWebMail is intended to check messages that arrive without going through the system MTA (Mail Transfer Agent) - such as POP3 messages.

Spam Check Options reference chart:

Term/Setting What it Sets/Changes
Messages to be checked
 This option specifies the mails to be checked by the spam check program.
Maximum size of the checked messages
 To eliminate system load, messages larger than this size won't be passed to the spam check program (in KB).
Treshold to determine Spam
 The spam check program is assumed to return a spam level of a message and messages with spam level equal or larger than this threshold willbe treated as spam.
