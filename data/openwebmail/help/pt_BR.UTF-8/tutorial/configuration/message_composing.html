OpenWebMail Ajuda e Tutorial >> Opções Criando/Enviando Mensagens

Em Criando/Enviando Mensagens se encontram as opções para edição e envio de mensagens.

Referência rápida para a configuração de Opções Criando/Enviando Mensagens.

Ítem Descrição
Message format Selects the default format (html/plain text) for composing messages:
  • Auto selected: Message composing format is the same as the replied/forwarded message, or plain text for new messages.
  • Text: Message composing format is plain text.
  • HTML: Message composing format is html if the browser supports the html text area.
  • HTML and Text: Message composing format is html, but message is sent both in html and plain text version (what is displayed to the recipient depends on the receiving email client).
Área para edição de mensagens: Altera o tamanho do campo de edição das mensagens.
Posição do botão "Enviar"Define a localização do botão enviar em relação à área de edição da mensagem.
Reformata linhas ao responderReajusta comprimento de linhas de acordo com largura da área de edição, para mensagens em formato texto.
Encaminhar com a mensagem originalDefine a posição da mensagem original ao encaminhar a mensagem.
Insere assinatura antes da mensagem encaminhada Selecione para inserir a assinatura antes do texto da mensagem encaminhada.
Cópia automática Configura um email ou lista de emails para cópia automática de toda mensagem enviada.
Salva cópia das mensagens em ENVIADASMarcando a "CheckBox" faz com que uma cópia da mensagem seja salva na Pasta ENVIADAS.
Salva cópia na pasta atualFaz a cópia da mensagem enviada na pasta corrente (a não ser que seja a de Entrada ou Rascunho).
Codificação de caracteres para...Define o padrão de codificação de caracteres para a mensagem enviada.
