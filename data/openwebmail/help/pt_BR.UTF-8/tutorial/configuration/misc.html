OpenWebMail Ajuda e Tutorial >> Opções Diversas

As Opções Diversas são várias configurações que não pertencem a um grupo de configurações específico.

Figura de referência para a configuração de Opções Diversas:

Termo/Ajuste O Que Altera
Usar Expressão Regular na...Marcando a "CheckBox" permite usar uma Expressão Regular para Procuras e Regras de Filtro
Esconder Pastas de dados internoMarcando a "CheckBox" permite esconder Pastas de dados interno
Lista pastas por categoriaLista pastas de usuário por categoria na página de pastas. Você define a categoria nomeando as pastas como categoria-nomedapasta, separados pelo caracter ou string configurados (default é "-", "/" não é permitido.
Intervalo de atualizaçãoSeleciona o intervalo de tempo para atualização de e-mail
Som para notificação de nova...Seleciona o som a ser tocado quando chegar uma nova mensagem (Clique no Ícone com o alto falante para testar)
Janela para notificação de nova...Seleciona o tempo de exibição da janela de notificação de novas mensagens. (ZERO desativa a exibição)
Dicionário para verificar ortografiaSeleciona o dicionário padrão a ser usado quando for fazer correção ortográfica na mensagem.
Temporização para LixeiraSeleciona o número de dias que o e-mail será mantido na pasta "Lixeira". Depois do número especificado, a pasta "Lixeira" será esvaziada ("Para sempre" desabilita a temporização)
Tempo Limite da SessãoMaximo de tempo permitido que a sessão ficará aberta quando não existir ação alguma do usuário
