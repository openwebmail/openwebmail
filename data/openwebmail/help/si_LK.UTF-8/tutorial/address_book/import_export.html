OpenWebMail පාඩම් මාලාව (tutorials)>> ලිපින පොත් ආයාත කිරීම හා නිර්යාත කිරීම - Importing & Exporting Address Books

ඔබ කලින් සාදාගත් ලිපින පොතක් ඔබගේ පරිගනකයේ තිබේ නම්,  OpenWebMail ලිපින පොතට ලිපින ඇතුල් කිරීමට ඔබ කරදර විය යුතු නැත. ආයාත මෙවලම තිබෙන්නේ ඒ සදහාය. ආයාත මෙවලම් තීරු බොත්තම ( ) ක්ලික් කරන්න. එවිට ඔබව ලිපින පොත් importer වෙත යොමු කෙරෙනු ඇත. මෙම importer Outlook Express 5 (හෝ ඉහළ) සහ Netscape Mail 4.x ( "x" යනු උප-අනුවාදයයි) සමගද ගැලපේ. ඔබ කළ යුතු වන්නේ නොබැදි ඉ-තැපැල් සේාවාලාභියා(offline e-mail client) විවෘත කර,  CSV ගොනු (කොමාවකින් වෙන්කළ අගයන්) හෝ TXT ගොනු (ASCII පෙළ ගොනුව) භාවිතයෙන් එහි ඇති ලිපින පොත නිර්යාත කිරීමයි. මෙම උපදෙස් ලිපින පොත් Importer හිද දී ඇත. OpenWebMail ලිපින පොත ආයාත කිරීම සදහා  ඔබට නොබැදි ඉ-තැපැල් සේාවාලාභියාගෙන්    ලිපින පොත නිර්යාත කළ යුතුය (Please consult the documentation for your offline e-mail client for instructions on exporting its address book). ඔබලිපින පොත නියමිත ආකාරයට සුදුසු ගොනු හැඩතලයකින් නිර්යාත කළ පසු, එය OpenWebMail ලිපින පොතට ආයාත කළ හැකිය. මෙය කිරීමට,  "Browse" ක්ලික් කර, CSV හෝ TXT ගොනුව ඔබගේ පරිගනකයේ ඔබ සුරැකූ ස්ථානයෙන් විවෘත කර ගන්න. අදාල ගොනුව තෝරා ගත් පසු,  "Open" ක්ලික් කරන්න. අවසාන වශයෙන්,  "ආයාත" ක්ලික් කරන්න. එවිට ඔබගේ නොබැදි ඉ-තැපැල් සේාවාලාභියාගේ ලිපින පොතේ පිටපතක් OpenWebMail ලිපින පොතෙහි දැකගත හැකි වනු ඇත.

ඔබට ඉහත ක්‍රියාවට ප්‍රතිවිරුද්ධ ක්‍රිාවලිය වන, OpenWebMail ලිපින පොත නොබැදි ඉ-තැපැල් සේාවාලාභියාට නිර්යාත කිරීමද කළ හැක. "ලිපින පොත නර්යාත කරන්න" බොත්තම ( ) ක්ලික් කර මෙය සිදු කරන්න. It will redirect you to a page with your address book entries listed in plain text. ලිපින පොත සුරැකීමට File > Save As... තෝරන්න. Select Text File (*.txt) for the Save as Type. ඔබට දැන් නොබැදි ඉ-තැපැල් සේාවාලාභියා වෙත ලිපින ආයාත කළ හැකිය. text files න් ලිපින පොත් ආයාත කිරීමට උපදෙස් සදහා ඔබගේ නොබැදි ඉ-තැපැල් සේාවාලාභියා සදහා වූ documentation බලන්න.
