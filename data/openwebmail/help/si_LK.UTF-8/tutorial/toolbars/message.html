ඔබ ලිපියක් විවෘත කරන විටදී , නව මෙවලම් තීරුවක් දිස්වෙමින්, ඔබ සිටින එම විශේෂිත පිටුව සඳහා වඩා වැඩි වරණයන් ප්‍රමාණයක් ලබා දේ.

ලිපි මෙවලම් තීරුව:

ආශ්‍රේය සටහන(Reference Chart):

 

පදය (Term)/අයිකනය නම/විකල්ප පාඨය කරන ක්‍රියාව
පූර්ව පියවරට ආපසුඔබව පෙර සිටි තැපැල් පෙට්ටිය වෙතට ආපසු රැගෙන යයි.
නව ලිපියක්නව ලිපියක් ලිවීම සඳහා ඉඩකඩ ලබා දේ.*
පිළිතුරුලිපිය එවූ තැනැත්තාට පිළිතුරු යවයි.
සැමට පිළිතුරුයොමුකළ අයද ඇතුලුව ලිපිය එවූ සියල්ලන්ටම පිළිතුරු යවයි.
ඇමුණුමක් සේ යොමුකිරීමට ලිපිය ඇමුණුමක් සේ යොමුකිරීමට
මුද්‍රණයලිපිය මුද්‍රණය කළ හැකි ආකාරයට පෙන්වයි.

ආශ්‍රේය සටහන(Reference Chart):

පදය (Term)/අයිකනය නම/විකල්ප පාඨය කරන ක්‍රියාව
Import Address Book [sender]ලිපිය එවන්නාගේ ලිපිනය ලිපින පොතට එක් කරයි.
Block Senderපෙරනයක් එක් කර ලිපිය එවන්නා අහුරා තබයි.
Block SMTP Relay(ලිපිය එවූ සේවාදායකය අහුරා තබයි) Blocks the SMTP Relay of that message (the server that forwarded the message)

ලිපියට පහතින් දෙවන කුඩා මෙවලම් තීරුවක්ද දැකිය හැකිය:

"පූර්ව පියවරට ආපසු (Back to Previous) " ඉහත එක් වගුවක සටහන්ව ඇති ආකාරයට ක්‍රියා කරයි.  දකුණට සහ වමට ඇති ඊතල සමඟ ඇති "1" තැපැල් පෙට්ටිෙය් වන බහුවිධ ලිපි අතර ගමන් කිරීම සඳහා වේ. (වමට ඇති ඊතලය පෙර ලිපියට යාමටත් දකුණට ඇති ඊතලය ඊළඟ ලිපියට යාමටත්ය).  ඊතල අතර ඇති අංකය ඔබ දැන් බලමින් සිටින ලිපිය කී වැන්නද යන්න පෙන්වයි.(1 තැපැල් පෙට්ටිෙය් මුල්ම ලිපිය දක්වයි).

The drop menu in which the text  Select Stationery for Reply  is in, allows you to select what stationery you would like to use for a reply.  Open the menu to display a list of stationery you have made.  Once you have selected one, you will be able to reply to the message with the stationery you selected.  If you do not have any stationery set up, click the stationery button ( ) after which you will be able to edit and/or create new stationery.  For a definition of stationery see The Settings Toolbar .

* ඔබ කෙටුම්පත් (drafts) ෆෝල්ඩරය තුළ ඇති ලිපියක් බලමින් සිටින විටදී ( ඔබගෙ ෆෝල්ඩර කළමණාකරණය (Managing Your Folders) බලන්න ), මෙම බොත්තම නව ලිපියක් ආරම්භ කරනු වෙනුවට දැනට ඇති ලිපියේ සැකසුම් කරනු ඇත.
