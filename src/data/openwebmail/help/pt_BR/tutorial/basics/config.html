Open Webmail Ajuda e Tutorial >> Configurações Básicas

As Configurações Básicas permite que você altere várias preferências que seu e-mail necessita. Se você está se logando pela primeira vez no Open WebMail, você será redirecionado para a sua configuração de Preferências, porém você poderá altera-las posteriormente. Este tutorial cobre as configurações básicas para personalizar seu webmail.

Estas são as principais opções de personalização:

  • Informações Pessoais
  • Preferências de Visualização
  • Visualização de Pastas
  • Operações com Mensagens
  • Opções na Leitura de Mensagens
  • Opções Respondendo Mensagens
  • Opções Filtrando Mensagens
  • Opções do Livro de Endereços
  • Opções do Calendário
  • Opções do Disco Web
  • Opções Diversas

Para muitos usuários, o padrão de configurações é o suficiente. Se você não quiser alterar as configurações neste momento, então clique no botão salvar que está no final da página.

Seu Administrador já deve ter pré-configurado essas opções para você, contudo dependendo destas pré-configurações alguns dos itens acima poderão não estar disponíveis.
