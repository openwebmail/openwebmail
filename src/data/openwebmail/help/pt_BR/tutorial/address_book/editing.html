Open Webmail Ajuda e Tutorial >> Editando o Livro de Endereços

O Livro de endereços é um pequeno banco de dados onde você irá acrescentar os seus contatos mais frequêntes, ele lhe permitirá ter o acesso a esses dados de fomra rápida e eficiente.Todavia o livro de endereços pode se usado de duas formas diferentes. O modo de Edição,no qual é usado para incluir informações, apaga-las, edita-las. Outra forma de utilização da lista de endereços é quando você está editando uma mensagem e tem a possibilidade de clicar sobre o ícone ( )disponibilizando assim todos os seus endereços do seu livro, perceba que esta opção vale para todos os tipos de destinatários (Para, CC, BCC).

No exemplo acima aparece o menu de edição, acrescente o nome completo do seu contato, seu endereço eletrônico e no campo (Nota) você poderá adicionar lembretes sobre o que se trata este endereço, agora clique no botão (Adicionar/Modificar).

Uma vez que você já tenha todos os seus endereços do E-mail cadastrados, você poderá editá-los mais tarde. Para editar uma entrada, clique no campo Nome da entrada, então aparecerá na área de edição acima. Edite a entrada e assim que terminar as alterações clique no botão(Adicionar/Modificar).

Você poderá deletar as entradas com muita facilidade, pois basta clicar no botão (Apagar) para excluir os dados, se preferir pode-se também apagar todas as entradas de uma só vez clicando no ícone ( Apagar todos os endereços )( ) , o sistema fará uma pergunta para a confirmação da deleção das mensagens.

Você pode também agrupar endereços múltiplos em um único nome. A criação de grupos é muito útil para se enviar a mesma mensagem para vários usuários de uma única relação Para agrupar endereços múltiplos, clique sobre o ícone de grupo ( ). Uma nova janela aparecerá com todos seus endereços. Selecione todos os endereços que você deseja agrupar, e o clique em ( Continuar ). Então clique ( Adicionar/Modificar ).
