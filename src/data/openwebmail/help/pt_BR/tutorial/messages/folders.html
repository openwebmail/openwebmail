Open Webmail Ajuda e Tutorial >> Gerenciando suas Pastas

As pastas são ferramentas muito usadas para manter o sua caixa postal sempre organizada. Existem cinco pastas predefinidas pelo sistema e você poderá criar uma série de outras pastas para se organizar de maneira adequada ao seu perfil. As cinco pastas predefinidas são:
  • Entrada
  • Guardadas
  • Enviadas
  • Rascunho
  • Lixo

Na caixa de Entrada é onde ficam os seus e-mails que acabaram de chegar, a menos que o usuário tenha criado algum filtro nas mensagens. Veja Configurando Filtros para maiores informações de como filtrar mensagens.). Uma vez que você já leu sua mensagem é possível então aplicar algumas ações para melhor se organizar, a mensagem pode ser apagada movendo-a para a pasta Lixo ou para qualquer outra pasta criada no sistema onde você poderá acessá-la posteriormente. Porém se a mensagem for realmente descartada existem duas formas para fazer isto:

Você pode selecionar a mensagem que irá para o lixo clicando em sua "CheckBOx" ( )e clicando sobre o ícone de Lixo no menu de ferramentas. ( ). Clique em "OK" para confirmar a ação. A segunda maneira é clicando novamente na "CheckBox" da mensagem que foi selecionada para ser apagada e depois a selecione a pasta "Lixo" no menu de ferramentas no canto direito da tela, clicando sobre o botão "Mover". Ou se você preferir apagar definitivamente a mensagem, selecione no menu a opção --APAGAR--.

Todavia, se você achar necessário ler esta mensagem no fufuro, é possível move-la para uma outra pasta armazenado assim a sua mensagem. Por exemplo, se você recebe frequentemente e-mails relacionados ao seu serviço o mais recomendado seria criar uma pasta com o nome de "Trabalho" e mover todos os e-mails de serviço para esta pasta. Para criar uma pasta siga os passos:

  1. Clique no botão "Pastas" na barra de ferramenta ( )
  2. Na caixa de texto "Nome da Pasta", digite o nome da pasta que você deseja criar.
  3. Clique em "Adicionar".

A seleção de criação de pastas fica a seu critério, podendo ser relcionadas por assunto, remetente, destinatário, etc.

Você também poderá gerenciar as próprias pastas, clicando no ícone "( ) " as seguintes opções aparecerão relacionados as ações de manutenção das pastas.
Ação O que ela faz.
Marcar como LidoMarca as mensagens na pasta como já lidas
Verificar ìndiceCheca o número de mensagens contidas na pasta.
Refaz índiceReconta o número de mensagens contidas na pasta.
RenomearRenomeia a pasta
ApagarApaga a pasta

Você também pode navegar entre as pastas, basta clicar sobre o nome das pastas e será redirecionado. Outra opção interessante da manutenção das pastas é a opção de "Backup", em frente de cada nome de pasta será visualizado o ícone de "Baixar mensagem", esta função disponibilizará a opção de guardar as pastas no seu Disco Rígido local em forma de arquivo.
