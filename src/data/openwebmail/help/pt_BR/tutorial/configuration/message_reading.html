Open Webmail Ajuda e Tutorial >> Opção de Leitura da Mensagem

As configurações das opções de leitura das mensagens são bem simples, basta você aceitar ou não a configuração apenas clicando na caixa tipo "CheckBox" de cada item.

Figura para referência do menu de opções.

Termo/Ajustes Ação
Usar fonte de tamanho fixo... Fixa o tamanho da fonte em todo o corpo da mensagem, com isto se melhora a visualização dos e-mails.
Usar a imagem do Smiley ...Permite que a mensagem interprete expressões de emoções " ;-) :-( " e as troque pelo "Smiley" .
Desativar JavaScript dentro...Desabilita JavaScript™ dentro das mensagens.
Desabilita CGI embutido...Desabilita "Scripts" de CGI embutidos nas mensagens
Mostrar mensagens anexadas...Transforma as imagens recebidas no e-mail em links, agilizando assim o processo de visualização do texto.
