Open Source(Código Aberto)

Denominação inventada em março de 1998 que segue a liberação do Mozilla para descrever o software distribuídocom código fonte sob as licenças que garantem a qualquer um o direito de usar, modificar, eredistribuir livremente, o código. A intenção era poder vender as maneiras dos'hackers' de fazer o software evitando as conotações negativas do termo "software livre"..
