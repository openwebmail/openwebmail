CGI (Common Gateway Interface)

Abreviação de "Commom Gateway Inteface", é uma especificação para a transferência de informações entre o Servidor WWW para um programa em CGI. Um CGI pode ser escrito em C, Perl ou Java. Os programas CGI são amaneira mais comum para os servidores WWW interagirem dinamicamente com os usuários. Muitas página em HTML utilizam CGI para pegar os dados de um formulário após ser preenchido e submetido por um usuário.
