Javascript™

O Java Script é uma linguagem desenvolvida pela Netscape para auxiliar desenvolvedores de sites a criarem sites mais interativos. Embora esta linguagem compartilhe de muitas das características e das estruturas dalinguagem Java original, ela foi desenvolvida independentemente. O Java Script pode interagir com o código fonte do HTML permitindo incrementar conteúdo dinâmico a aplicação.O Javascript é endossado por um grande número de companhias do software e é umalinguagem aberta que permite que qualquer pessoa possa utilizá-la sem a necessidade da compra de uma licença. É suportado por browsers mais recentes da Netscape e da Microsoft, embora o Internet Explorer suporte somente um subconjunto, que a Microsoft chama de Jscript.
