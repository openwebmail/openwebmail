Codificação de Caracteres

É uma lista definida dos caracteres reconhecidos pelo software e pelo computador. Cadacaractere é representado por um número. O conjunto de caracteres do modo ASCII, por exemplo, usa os números 0 a 127 para representar todos oscaracteres ingleses e também caracteres especiais. O conjunto europeu de caracteres de padrão ISO é similar ao ASCII, mas contêmcaracteres adicionais para línguas européias.
