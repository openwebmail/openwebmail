Open Webmail 使用指南 >> 导入/导出地址簿

你也许已经储存了一份地址簿在你的电脑中，而不想在Open Webmail的地址簿中从头新增这些资料。这时候就是该使用“导入”工具了，点选导入工具条按钮 ( ) ，系统将会开启地址簿导入程序画面。导入程序相容于Outlook Express 5（或更新的版本），以及Netscape Mail 4.x（“x”表示次版本)。你需要开启你的离线式邮件阅读程序，导出它的地址簿，存成 CSV (Comma Separated Values) 或是 TXT（ASCII Text File）格式。导入程序画面也提供了简单的操作步骤。你必须从你的离线式邮件阅读程序中导出地址簿，以便于导入到Open Webmail的地址簿中。（请参考你的离线式邮件阅读程序的使用手册以了解如何导出地址簿）。当你正确地导出你的地址簿并且存成适当的格式，你就可以开始使用Open Webmail来导入地址簿了。要开始导入地址簿，请点选“浏览”按钮，并且从先前储存在电脑中的目录中开启CSV或是TXT格式的文件，当你找到该适当的文件时，接著点选“打开”。最后，点选“导入”按钮。此时Open Webmail上的地址簿内容就会和你的离线式邮件阅读程序中的地址簿内容完全一样了。

你也可以相反地操作，从Open Webmail中导出地址簿，从离线式邮件阅读程序中导入地址簿。点选“导出地址簿”按钮（ ），它会引导出一个文件下载窗口，选择保存按钮以保存文件，依据你的需要输入文件名称，存档类型请选择Microsoft Excel Comma Separated Values File。你现在可以从你的离线式邮件阅读程序中导入地址簿了，关于从文字文件导入地址簿的操作步骤请参考你的离线式邮件阅读程序手册。
