Tutorial Bantuan Open Webmail >> Konfigurasi Pengaturan Dasar

Konfigurasi Pengaturan Dasar mengijinkan anda untuk mengubah berbagai pilihan yang sesuai dengan kebutuhan email anda. Jika anda baru pertama kali menjadi pengguna Open Webmail, anda akan diarahkan ke control panel konfigurasi pengaturan anda, daripada Folder Email Baru anda. Tetapi, anda mungkin ingin membuat webmail anda sehingga dapat digunakan dan dapat menjalankannya dengan secepat mungkin. Tutorial ini akan meliputi hal-hal dasar untuk mengkonfigurasikan pengaturan anda.

Ini adalah bagian yang utama di dalam konfigurasi pengaturan:

  • Informasi Pribadi
  • Pilihan Tampilan
  • Pilihan Membaca Pesan
  • Operasi Pesan
  • Pilihan Membalas Pesan
  • Pilihan Mem-filter Pesan
  • Pilihan Kalender
  • Pilihan Lain-lain

Bagi banyak pengguna, default pengaturan ini dirasakan sangat cukup. Jika anda tidak ingin mengubah pengaturan pada saat ini, klik tombol simpan pada bagian bawah di halaman ini.

Sistem Administrator anda mungkin akan mengkonfigurasikan ulang atau menghapuskan pilihan ini untuk anda.
