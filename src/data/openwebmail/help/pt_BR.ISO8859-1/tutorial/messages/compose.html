Quando você compõe uma mensagem, você sabe pra quem você está enviando, qual é o assunto, e, com certeza, o que você quer escrever na mensagem.

Para iniciar uma mensagem, clique no ícone "Compor Mensagem" na barra de ferramentas (veja Usando as Barras de Ferramentas ). Você será redirecionado para uma página como essa:

Esta tela que você verá quando estiver compondo uma mensagem. Clique para ver mais detalhes.

Há algumas opções que você pode querer considerar. Se você selecionar a opção "Salva Cópias em Enviadas", você manterá uma cópia da mensagem na pasta Enviadas. Desta maneira, você pode manter um registro das mensagens enviadas também. Selecionando a opção "Confirmação de Leitura", você solicitará uma confirmação de leitura para o destinatário.
