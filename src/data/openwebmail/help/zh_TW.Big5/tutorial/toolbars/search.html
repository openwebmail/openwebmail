Open Webmail 說明指南 >> 搜尋工具列

搜尋工具列允許你搜尋目前所在資料夾的訊息。

使用這個工具列真的十分簡單。 這個雙向左箭頭 ( )  將會完全跳到目前所在資料夾的第一筆訊息。請看 顯示參考 來改變顯示在每一頁的訊息號碼。

向左的箭頭 ( ) 將會跳到訊息的前一頁。

文字欄位 "1"，告訴你目前是在總頁數當中的第幾頁. 你可以在目錄夾中改變數字來跳到訊息的任何一頁. 例如,如果寫 "[Page 2 of 7]",你可以鍵入 "5"來代替 "2"，就會跳到第五頁了。

向右箭,以及雙向右箭完全和向左箭及雙向左箭相反。 

你可以搜尋目前資料夾的訊息，藉由按鈕"搜尋"旁邊的文字欄位鍵入欲搜尋的字串，並按下搜尋. 你針對這封訊息是 從哪裡來的、誰發出的、主旨、日期、附加的檔案名稱、信件表頭、訊息的文字內容 或以上所有 等條件進行搜尋。

這個工具列是用於在目前的信件匣或是資料夾中進行基本的郵件搜尋和查看。另外還有一個較進階的工具列是用來進行更細部的搜尋。請查看 進階郵件搜尋 來學習如何執行特殊的搜尋要求。
