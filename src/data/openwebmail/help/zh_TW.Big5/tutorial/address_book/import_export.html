Open Webmail 說明指南 >> 匯入與匯出通訊錄

你也許已經儲存了一份通訊錄在你的電腦中，而不想在 Open Webmail 的通訊錄中從頭新增這些資料。這時候就是該使用 "匯入" 工具了，點選 匯入工具列按鈕 ( ) ，系統將會開啟通訊錄匯入程式畫面。匯入程式相容於 Outlook Express 5 (或更新的版本)，以及Netscape Mail 4.x ( "x" 表示次版本)。你需要開啟你的離線式郵件閱讀程式，匯出它的通訊錄，存成 CSV (Comma Separated Values) 或是 TXT(ASCII Text File)格式。匯入程式畫面也提供了簡單的操步驟。 你必須從你的離線式郵件閱讀程式中匯出通訊錄，以便於匯入到 Open Webmail 的通訊錄中。 (請參考你的離線式郵件閱讀程式的使用手冊以了解如何匯出通訊錄)。當你正確地匯出你的通訊錄並且存成適當的格式，你就可以開始使用 Open Webmail 來匯入通訊錄了。要開始匯入通訊錄，請點選 "瀏覽" 按鈕，並且從先前儲存在電腦中的目錄中開啟CSV 或是 TXT 格式的檔案，當你找到該適當的檔案時，接著點選 "開啟"。最後，點選  "匯入" 按鈕。此時 Open Webmail 上的通訊錄內容就會和你的離線式郵件閱讀程式中的通訊錄內容完全一樣了。

你也可以相反地操作，從 Open Webmail 中匯出通訊錄，從離線式郵件閱讀程式中匯入通訊錄。點選"匯出通訊錄" 按鈕 ( )，它會引導出一個檔案下載視窗，選擇儲存按鈕以儲存檔案，依據你的需要輸入檔案名稱，存檔類型請選擇 MicrosoftExcel Comma Separated Values File。你現在可以從你的離線式郵件閱讀程式中匯入通訊錄了，關於從文字檔案匯入通訊錄的操作步驟請參考你的離線式郵件閱讀程式手冊。
