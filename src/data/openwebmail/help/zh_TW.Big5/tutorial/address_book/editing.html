Open Webmail 說明指南 >> 編輯通訊錄

通訊錄是一個你常用郵件位址的小型的資料庫，可以讓你快速且容易地存取它。然而，通訊錄有兩種不同的使用模式。編輯模式，用來新增一筆紀錄，刪除，編輯，以及使用進階功能。另外一種就是所謂的使用模式，可以在外寄郵件上立即地加入收信人的資訊(例如：收件者，CC 副本抄送，BCC 密件副本)。

當然，首先你需要先新增紀錄。你必須先開啟通訊錄來編輯。只要在主要工具列點一下 ( ) 這個圖示，就可以開啟通訊錄了。你會被導引到通訊錄的編輯畫面。

很明顯地，你的第一步就是要新增一筆紀錄。 (注意: 也許會出現些由系統管理者設定的一些紀錄)在Name 文字欄位上輕點一下，並且輸入必須的資料。使用 "Tab" 鍵可以直接跳到下一欄。你也可以輸入暱稱到 Note 欄位。當你輸入完成後，點選 Add/Modify按鈕，你現在就可以快速地使用這一筆郵件位址來寄信了。

當你完成所有郵件地址的建檔後，也許會想要修正些資料。如想要編輯郵件地址資料，可以在名稱欄位上點一下，那一筆資料就會出現在上方的編輯區域。就可以修改該筆資料，改成你要的樣子。當你修改完成後，再點選一次 Add/Modify 按鈕就可以了。

你可以直接點選右方的 Delete 按鈕，刪除你想要刪除的項目。 你也許也想要刪除所有的項目，但是分開一次一次刪除會是枯燥無味的事。當然，你也可以點選  按鈕，直接 "清除所有項目"。系統會要求你確認，點選 "OK" 就會刪除所有項目、或是點選 "Cancel" 就會取消目前的操作。

你也可以把許多郵件地址加入到一個群組。使用一個郵件地址項目就可以讓你寄信給許多已經存在的郵件地址，不需要一筆一筆地重新輸入。 想要把許多郵件地址加入到群組中，點選群組 ( ) 就可以了。你會被導引到你的所有郵件地址的新視窗中，點選那些你希望加入到群組的項目，然後點選繼續按鈕。最後完成時再點選 Add/Modify 。
