Open Webmail 說明指南 >> 郵件過濾設定

郵件過濾設定決定如何過濾郵件。

郵件過濾設定參考表：

項目/設定 說明/改變
重複信件上限如果有一群信件有相同的寄件者與標題，且數目超過這個上限時，則這群信件會被丟到垃圾桶
過濾偽造來源端的信件如果一封信件是轉自一個偽造來源端位址的 SMTP 主機，則這封信件會被丟到垃圾桶
過濾偽造寄件人地址的信件如果一封信件被發現偽造其寄件者地址，則這封信件會被丟到垃圾桶
過濾偽造 EXE 附件型態的信件如果一封信件含有任何可執行型態的附件 (exe, com, bat, scr, vbs...)，而這些附件又嘗試偽裝其附件型態的話，則這封信件會被丟到垃圾桶
