Bienvenue dans le tutoriel Open Webmail!   Open Webmail est un projet open source libre de droits, qui permet aux utilisateursde consulter leur messagerie électronique depuis n'importe où dans le monde! Vous pouvez utiliser Open Webmail pour consulter et envoyer vos messages en voyage, ou bien en remplacement de votre logiciel de messagerie "hors-ligne". Open Webmail peut également vous aider dans la gestion de votre activité quotidienne à l'aide d'un outil de calendrier intégré. Dans ce tutoriel, vous trouverez des exemples, schémas, et définitions, qui vous permettront de tirer le meilleur parti d'Open Webmail...
