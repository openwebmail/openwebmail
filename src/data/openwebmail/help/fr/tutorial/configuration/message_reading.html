Tutoriel d'aide Open Webmail >> Options de lecture des messages

Les options de lecture des messages sont simplement des options de type oui-ou-non qui modifient la façon dont les messages reçus sont affichés. Vous pouvez modifier ces options selonvos préférences, et en particulier pour mieux profiter de la taille de votre écran, si vous le souhaitez.

Eléments de Référence:

Terme/Réglage Ce qui est positionné/modifié
Utiliser une police de taille fixe pour la lecture des messagesFixe la largeur de la police, pour mieux s'harmoniseravec la taille de l'écran
Utiliser des 'Smileys' lors de la lecture des messagesPermet l'utilisation d'images pour l'affichage des smileys (Egalement appelés "Emoticons") au sein des messages.
Interdire le JavaScript dans les messagesDésactive le JavaScript™ dans les messages.
Désactiver les CGI inclus dans les messagesDésactive les scripts CGI dans les messages
Afficher les images attachées comme des liensUtilise un lien pour les images attachées, plutôt que de les afficher directementdans l'E-mail.
Jeu de caractères pour la réponseCeci permet de sélectionner le Jeu de caractères pour les messages sortants.
