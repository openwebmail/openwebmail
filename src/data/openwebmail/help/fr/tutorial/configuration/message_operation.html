Tutoriel d'aide Open Webmail >> Utilisation des messages

Les options de réglage de l'utilisation des messages permettent de préciser les actions à effectuer sur les messages (comme Déplacer et Copier).

Eléments de référence:

Terme/Réglage

Ce qui est positionné/modifié
Répertoire de destination par défautPrécise le dossier de destination par défaut des messages non filtrés.
Confirmer avant de déplacer/copier un messageDemande une confirmation avant de déplacer ou copier un message vers un autre dossier.
Afficher le message suivant après un "déplacer/copier"Affiche le message suivant du dossier courant après un déplacer/copier.
Rappatriement automatique des mails POP3 à l'identification (Login)Rappatrie vos E-mails sur le serveur Open Webmail dès lors que vous vous connectez (reçoit tous les E-mails qui vous ont été éventuellement envoyés pendant votre période de déconnexion du serveur).
Déplacer les messages lus dans "Enregistrés" à la déconnexionDéplace tous les anciens messages (messages lus) vers le dossier "Enregistrés" lors de votre déconnexion d'Open Webmail
