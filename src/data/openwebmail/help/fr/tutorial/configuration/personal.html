Tutoriel d'aide Open Webmail >> Réglages personnalisés

Les Informations personnelles comprennent des éléments que vos correspondants verront lorsqu'ils recevront des E-mails de votre part, ainsi que d'autres préférences personnelles.

Veuillez consulter les éléments suivants pour effectuer la configuration de vos réglages personnels:

Terme/Réglage Ce qui est positionné/modifié
LangueSpécifie la langue à utiliser pour l'affichage d'Open Webmail
Jeu de caractèresModifie le jeu de caractères pour les E-mails (Il est recommandé de laisser la valeur par défaut).
Fuseau horairePermet de préciser le décalage horaire par rapport au méridiende Greenwich. Cliquez sur l'icône "Terre/horloge" pour visualiser une carte indiquant l'affectation des fuseaux horairesaux différents pays.
Faire suivre TOUS vos messages àPermet de faire suivre la totalité des messages reçus survotre adresse E-mail vers une autre adresse. La case à cocher située juste à coté permet de préciser si vous souhaitez conserver un duplicata de ces messages dans votre boîte aux lettres.
ExpéditeurL'affichage que le destinataire de votre message verra dans le champ "Expéditeur" de l'En-tête du message.
Répondre àL'adresse à laquelle le destinataire répondra (s'il utilise la fonction "Répondre")
Réponse automatiqueUne réponse automatique à la réception de vos E-mails.
SujetLe sujet de votre réponse automatique (valeur par défaut recommandée).
Texte de la réponse automatiqueLe message de la réponse automatique (maximum 500 caractères).
SignatureUne signature d'E-mail qui sera ajoutée à la fin de chacun des messages que vous rédigerez.
