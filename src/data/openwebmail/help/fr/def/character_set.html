Jeu de caractères

Il s'agit d'une liste de caractères reconnus par le matériel et le logiciel de l'ordinateur. Chaquecaractère est représenté par un nombre. Le jeu de caractères ASCII, par exemple, utilise les nombres 0à 127 pour représenter tous les caractères de l'anglais, ainsi que quelques caractères de contrôle. Le jeude caractères Européen ISO est similaire à l'ASCII, mais contient des caractères additionnels, pourcorrespondre à d'autres langages Européens (ajout des caractères accentués par exemple).
