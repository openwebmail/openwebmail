URL (Uniform Resource Locator)

Abréviation de "Uniform Resource Locator", il s'agit d'un format d'adresse qui identifie un document ou une ressource sur le World Wide Web (WWW). Cette définition est là seulement pour enregistrer le fait que le terme est utilisé et prononcé en tant que son abréviation, c'est à dire "U-R-L".
