Tutorial Bantuan Open Webmail >> Menyimpan Pesan

Anda mungkin ingin mengetikkan pesan, dan mengirimkannya kemudian. Hal ini dapat dilakukan apabila anda menyimpannya dalam folder Simpan Catatan. Ini menyimpan semua informasi catatan atas lengkap (Kepada, Dari, Judul, dan lain-lain) sama seperti teks pesan. Sekali anda sudah selesai mengetik pesan anda, klik tombol "Simpan Catatan", dan pesan anda akan disimpan dalam folder "Simpan Catatan". Untuk mengembalikan pesan anda dan mengirimkankan kemudian, temukanlah di dalam folder "Simpan Catatan"anda, (lihat bagian Mengatur Folder Anda), bukalah, dan klik tombol "Kirim". Anda juga dapat mengubahnya sebelum anda mengirimkan pesan tersebut.
