Javascript™

Bahasa script yang dikembangkan oleh Netscape untuk memungkinkan pengarang Web untuk mendesain site interaktif. Meskipun ada banyak fitur dan struktur dari bahasa Java yang lengkap, ini dapat dikembangkan secara bebas. Javascript dapat berinteraksi dengan sumber kode HTML, yang memungkinkan pengarang Web untuk memperindah website mereka dengan isi yang dinamis. JavaScript didukung oleh berbagai perusahaan software dan suatu bahasa yang terbuka yang dapat digunakan oleh semua orang tanpa perlu untuk membeli ijin untuk menggunakannya. Ini didukung dengan browser terbaru dari Netscape dan Microsoft, meskipun Internet Explorer hanya mendukung subset, yang disebut Microsoft sebagai Jscript.
