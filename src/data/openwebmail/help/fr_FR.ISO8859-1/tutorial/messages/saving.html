Tutoriel d'aide Open Webmail >> Enregistrer un message

Vous pouvez vouloir commencer la frappe d'un message pour l'envoyer plus tard. Ceci est possible en enregistrant un Brouillon. Toutes les informations d'En-tête d'E-mail (Destinataire, Expéditeur, Sujet, etc.) seront enregistrées, ainsi que le texte du message. Une fois que vous aurez terminé votre saisie, cliquez sur le bouton "Enregistrer brouillon", et votre message sera stocké dans le dossier "Brouillons". Pour reprendre votre message, et l'envoyer ensuite, sélectionnez le dans votre dossier "Brouillons", (voir Gérer vos dossiers ), ouvrez-le, et cliquez sur "Envoyer", après l'avoir éventuellement complété.
