Open Webmail Help Tutorials >> The Search Toolbar

The search toolbar allows you to search the messages in the current folder.

Using this toolbar is actually quite simple. The Left Arrow () will jump to the previous page of messages. The Right Arrow () to the next page. See Display Preferences to change the number of messages that are displayed per page.

The numbers in between the arrows tells you what page you are viewing (1 specifies the 1st page in the folder) and the total pages (2) in the folder.

In the drop menu Page you can change this to jump to any page of messages in the folder. Clicking the drop menu gives you the total list of pages. For example, choose "5" and your jumping to page five. The + sign lets you quickly display a maximum of 1000 messages on one page.

You can search the messages in the current folder by simply typing in the search query in the text field next to "Search" and selecting the criteria to search in. You can search the who the letter is From, who it was To, the Subject, the Date, the Attachment Filename, All of the Above (Header), the Content of the message or All of the Above (All). Once you have selected all of your search criteria, click the "Search" button or hit "Enter" on the keyboard. The Clear button removes the search query from the field and all messages are displayed again.

This toolbar is for basic message searching and browsing within the current mailbox or folder.  There is an advanced search toolbar for more detailed searches.  Please see Advanced Message Search to learn how to perform for specific search queries.
